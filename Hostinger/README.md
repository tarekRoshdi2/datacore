# 📁 مجلد حزمة نشر Hostinger (Hostinger Deployment Bundle)

يحتوي هذا المجلد على ملفات التكوين والإعدادات السريعة المخصصة لاستضافة **Hostinger**.

## 📄 محتويات المجلد:
1. **`ecosystem.config.cjs`**: ملف إعدادات **PM2** لتشغيل وإدارة سيرفر التطبيق في الخلفية.
2. **`.env.example`**: نموذج للمتغيرات البيئية المطلوبة لعمل التطبيق على Hostinger.
3. **`.htaccess`**: ملف إعداد خادم Apache على Hostinger لضبط المسارات والحماية.
